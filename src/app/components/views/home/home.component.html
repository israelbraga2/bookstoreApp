Bem vindo! Sistema para exemplificar um CRUD usando Angular e Spring Boot

My Book Store

BACK END: Criamos uma API Rest usando Java com Spring Boot e também usamos JPA com Hibernate para persistir as informações no banco H2. Aprendemos também a validação de dados, tratamento de exceções, uso adequado do protocolo HTTP no padrão REST, foi desenvolvido um frontend usando o Angular , consumimos nossa API REST usando nosso frontend. Biblioteca completa com CRUD (CREATE, READ, UPDATE e DELETE) gerenciando seus livros e categorias via requisições web. API REST foi implantada na NUVEM da HEROKU !!! Nosso front-end foi implementado no Github Pages !!!
